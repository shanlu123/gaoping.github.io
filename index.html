7-22 堆栈模拟队列 (25point(s))

发表于 2021-01-30 分类于 PTA刷题 阅读次数： Valine： 本文字数： 3.1k 阅读时长 ≈ 3 分钟

题目

解题思路

  1. 首先比较两个堆栈的大小，小的栈为输入栈，大的栈为输出栈
  2. 如果输入栈满了，而输出栈为空，则将输入栈中的所有元素全部转移到输出栈中（如果还有输入这时就不能再转移了）
  3. 如果输入栈满了，而输出栈不为空，则此时栈满（因此可以得知整个栈的最大容量为输入栈的两倍）
  4. 输出：
    1. 输出栈不为空时，则输出栈的栈顶元素为队头元素
    2. 输出栈为空，而输入栈不为空时，则将输入栈中的元素全部转入输出栈中，再输出栈顶元素
    3. 如果输出栈和输入栈中的元素均为空时，则栈空

思路

  2. 后面的数是正数，加上它就一定比当前的结果大，更新当前求出的结果。否则不更新当前的结果值

PAT B1020月饼 (25*point(s))

发表于 2021-01-04 分类于 OJ--《算法笔记》 阅读次数： Valine： 本文字数： 3.4k 阅读时长 ≈ 3 分钟

题目

思路

  1. 这里采用“总是选择单价最高的月饼出售，可以获得最大的利润”的策略
  2. 因此，对每种月饼，都根据其库存量和总售价来计算出该种月饼的单价，之后将所有月饼按单价从高到低排序。
  3. 从单价高的月饼开始枚举，①如果该种月饼的库存量小于需求量，则将该种月饼全部卖出，此时需求量减去该种月饼的库存量大小，收益增加该种月饼的总售价大小。②如果该种月饼的库存量大于等于需求量，则只提供需求量大小的月饼，此时收益值增加当前需求量乘以该种月饼的单价，而需求量减为零。
  4. 这样，最后得到的收益值即为所求的最大收益值。
